# Ad Scripts

Use this file when the user needs Meta ads, UGC structures, or short-form creative angles.

## Shared Rules

- open with the problem
- show use in the first 3 seconds
- keep the script under 30 seconds
- use on-screen captions
- avoid claims the product cannot support
