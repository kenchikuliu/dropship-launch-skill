# 7-Day Test Plan

Use this file when the user wants a first validation plan.

## Budget

- total first test: `GBP 210` to `GBP 315`
- daily budget: `GBP 10` to `GBP 15` per ad set

If budget is tight, launch one product only.

## Default Campaign Setup

- market: United Kingdom
- objective: sales
- placement: broad placements are fine for first pass
- optimization: purchase

## Product Start Prices

- hidden drawer: `GBP 29.99`
- under-sink organizer: `GBP 34.99`
- pet hair remover: `GBP 24.99`

## Metrics

Practical targets:

- link CTR above `1.5%`
- CPC below `GBP 1.20`
- add-to-cart rate above `4%`
- checkout rate above `2%`

Use these as operating thresholds, not universal truths.

## Kill Rules

Kill a creative if:

- spend reaches about one selling price with no add-to-cart
- CTR is weak after enough impressions
- comments show a trust or product issue

Kill a product faster than a founder usually wants to. Weak products waste more than weak ads.

## Scale Rules

Scale only when:

- at least 2 purchases exist
- CPA is workable
- comments are healthy

Increase budget by `20%` to `30%`, not a sudden jump.

## Day Flow

### Day 0

- finish page
- verify tracking
- test checkout

### Day 1

- launch product one
- monitor spend and comments

### Day 2

- kill weak creatives

### Day 3

- review page behavior

### Day 4

- duplicate strongest angle with a new opener

### Day 5

- decide if the problem is product or creative

### Day 6

- test one offer or one price adjustment if signal exists

### Day 7

- decide to kill, continue, or scale
